# Инструкция по установке

## Быстрый старт

```bash
git clone https://github.com/nj7wrr4fz5-cloud/NEURAL_ARCHITECT_PREMIUM_PLUS_PLUS.git
cd NEURAL_ARCHITECT_PREMIUM_PLUS_PLUS
npm install
npm run dev
```

Откройте http://localhost:5173

## Деплой на Vercel

1. Перейдите на https://vercel.com
2. New Project -> выберите репозиторий
3. Deploy

## Сборка

```bash
npm run build
```

Папка dist готова для хостинга.
